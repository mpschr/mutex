# MutEx
a simple algorithm to estimate the significance of a mutual exclusive pattern or co-occurrence

![the MutEx algorithm](mutex-algorithm.png)


## dependencies

 - scipy 0.17 or higher
 - pandas 0.17 or higher
 - numpy 1.10.4

 - joblib
 - multiprocessing

## provenance
